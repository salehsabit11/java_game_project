# Flappy Elephant 
You might think "Can an elephant FLY??". We say "Why not!". A simple Flappy Bird clone game based on java, implimenting all concept of OOP.

## Getting Started
**Prerequisites**
- Java JDK 8 or higher
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code) or terminal

### Run the Game

**Using terminal**
```
javac *.java
java App
```
**Using IDE:** 

Open the project and Run `App.java`
### How to play

Simply just press `Space` bar to flap/jump

- Avoid Pipe
- Dont hit the ground or celling
